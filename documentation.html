AddrSearch API Documentation

Introduction

AddrSearch is a cloud-hosted Australian address lookup and validation service built on the authoritative Geocoded National Address File (G-NAF). It supports fuzzy address search, G-NAF validated address results, and postcode lookup. The API is designed for developers and businesses looking for reliable, scalable address solutions.

Base URL

https://api.addrsearch.com.au

Authentication

All requests must include an API key as a query parameter:

?key=YOUR_API_KEY

API Endpoints

/search

Returns a list of addresses based on fuzzy matching. This is useful for handling misspellings, partial inputs, and free-text entries.

Method: GET

Example: /search?address=53+Cardinal+Avenue+Beecroft+NSW&key=YOUR_API_KEY

/gnaf

Returns structured G-NAF validated address data. Ideal for applications that require the most accurate and official address data.

Method: GET

Example: /gnaf?address=53+Cardinal+Avenue+Beecroft+NSW&key=YOUR_API_KEY

/postcode

RLooks up suburbs and locations based on postcode input.

Method: GET

Example: /postcode?code=2159&key=YOUR_API_KEY

Notes and Limitations

  • PO Box support is not available at this time but is part of the roadmap.
  • Results are limited to 20 matches per query.
  • The data is updated quarterly from official G-NAF releases.
