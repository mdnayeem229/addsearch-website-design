Core Features

Fuzzy Address Search

Handles misspellings, incomplete entries, and typos. Example:

  • 53 Cardnal Av, Beecroft' → '53 Cardinal Avenue, Beecroft NSW 2119' Use Cases
  • Address autocomplete in forms
  • Mobile checkout optimisation
  • CRM data cleanup

G-NAF Address Lookup

Validated, structured addresses from the official Geocoded National Address File (G-NAF). Returns:

  • Lot/unit details
  • Street and locality
  • State and postcode
  • Latitude and longitude (geocoded) Ideal for
  • Government and utilities
  • Health and logistics platforms
  • Compliance and identity verification

Postcode Lookup

Search by suburb or postcode to retrieve all valid matches. Examples:

  • 2119' → Beecroft, Cheltenham
  • Beecroft' → 2119 (NSW) Use Cases
  • Service coverage validation
  • Auto-filling region data
  • Demographic analysis
